## Project Description
This is a PHP-based student portal that allows students to register, log in, select courses (maximum 10), upload a passport photo (3MB limit), and view generated results. The system uses session management and does not use a database.

## Features
- User Registration
- Login & Logout (Session Management)
- Course Registration (Max 10)
- Passport Upload (3MB limit)
- Dashboard Display
- Statement of Results Generation

## Technologies Used
- PHP
- HTML5
- CSS3
- Sessions (No Database)
